## 2. Push your changes to github 
Inside your VsCode terminal run these command below:

Stage everything:
```bash
git add .
```

Commit with a short message, change the "Add new feature" to upload your message:
```bash
git commit -m "Add new feature"
```

Push to github
```bash
git push
```
